INSTAL PHALCON PHP

- DOWNLOAD PHALCON DLL: https://phalcon.io/en-us/download/windows

- DOWNLOAD PSR DLL: https://pecl.php.net/package/psr/1.0.0/windows

- INSTALAR php_phalcon.dll NO SERVIDOR LOCAL XAMPP - PHP 7.2

	- C:\xampp\php72\ext
		- COLAR AS DLLs: php_phalcon.dll E php_psr.dll

	- C:\xampp\php72
		- NO FINAL DO ARQUIVO INSERIR: extension=php_psr.dll
		- NO FINAL DO ARQUIVO INSERIR: extension=php_phalcon.dll

- INSTALAR O Phalcon Devtools (GLOBAL - INSTALAÇÃO NA PASTA AppData DO WINDOWS)
	- composer global require phalcon/devtools
	
- INSTALAR O Phalcon Devtools (APENAS EM SEU PROJETO)
		- phalcon project NOME_DO_PROJETO --type=simple (CRIAR PROJETO TEMPLATE ENGINE PHTML)
		- phalcon project sistema-msds --type=simple --template-engine=volt (CRIAR PROJETO TEMPLATE ENGINE VOLT)
		- composer require phalcon/devtools

- CRIAR O PROJETO
	- phalcon project NOME_DO_PROJETO --type=simple (CRIAR PROJETO TEMPLATE ENGINE PHTML)
	- phalcon project sistema-msds --type=simple --template-engine=volt (CRIAR PROJETO TEMPLATE ENGINE VOLT)
	
- EXECUTAR O PROJETO PHALCON ATRAVÉS DE SEU PRÓPRIO WEBSERVICE
	- ACESSAR A PASTA DO PROJETO E EXECUTAR O COMANDO: $(which php) -S localhost:8000 -t public .htrouter.php
	
- CRIAR AS Models ATRAVÉS DO BANCO DE DADOS
	- CRIAR O BANCO NO HeidiSQL / PHP My Admin / MySQL Workbanch ...
	- CONFIGURAR O ARQUIVO CONFIG EM: APP/CONFIG/config.php
		'host'        => 'localhost', //INFORMAR O HOST DO BANCO
        'username'    => 'root',	  //INFORMAR O NOME DE USUÁRIO DO BANCO
        'password'    => '', 		  //INFORMAR A SENHA DO BANCO
        'dbname'      => 'test',	  //INFORMAR O NOME DO BANCO
	
	- ACESSAR A PASTA DO PROJETO E EXECUTAR O COMANDO: phalcon all-models OU phalcon all-models --namespace=Models --extends=Model --force
